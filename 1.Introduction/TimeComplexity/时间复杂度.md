## 代码分析

```c
void analysis(int n)
{
  int i,j;
  int count = 0;
  int num = 1;
  //以n = 5 进行分析
  //外层循环执行5次  对应的i 为 {0,1,2,3,4}
  for (i = 0; i < n; i++)
    //内存循环 根据(j = i; j < n) 内存循环里面语句执行的次数对应为{5,4,3,2,1}
    for (j = i; j < n; j++) {
          num = num * 2;
          count++;
    }
  printf("执行次数:%d",count);
}
```

**总结**

**外层循环**的时间复杂度为`n`,**内层循环**的时间复杂度为`(n+1)/2`,根据乘法法则，程序的时间复杂度为`n(n+1)/2`
